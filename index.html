Tony Cheng¶

Introduction¶

Whether you have you are a new or old fan of Pokémon, you know that the Pokémon universe is constantly changing and expanding. So in order to better understand the Pokémon universe, we will be exploring Pokémon data from all seven Pokémon generations! That's 802 Pokémon plus their variations, such as mega ecolutions, Alolan versions, and primal forms! But don't let those numbers scare you away. This tutorial will show you how to format data, choose your starter Pokémon, analyze and graph type advantages of different Pokémon, find the strongest Pokémon, and better understand each Pokémon generation, so you can become the very best like no one ever was.

def effectiveness(attacker,defender):
   return typeChart[typeMap[defender]][typeMap[attacker]]

This may look very confusing at first, but it is based on simple rules. As was stated before, some Pokémon match up better against certain types. The effectiveness function finds how effective an attack of a certain type is against another Pokémon type. To clarify the results, 0 means no effect, 0.5 is half damage, 1.0 is normal/neutral damage, and 2.0 is double damage. Here is an example.

In [9]:

Using the effectiveness function, the type effectiveness potential can be calculated based on all Pokémon in its deneration. You can make a simplified version that adds points per effectiveness of type without considering the second type. This will make it easier to calculate while also still giving you a good idea of a Pokémon's type effectiveness.

In [10]:
gen1 = Pokemon[Pokemon['GENERATION']==1]  #Generation 1 Pokemon

def effects(type1,type2):
    result = {}
    for index, row in Pokemon.iterrows():
        effect = effectiveness(type1,row['TYPE 1'])
        if not pd.isnull(row['TYPE 2']): #checks for a secondary typing
            effect += effectiveness(type1,row['TYPE 2'])
        if (type2 != ''):
            effect += effectiveness(type2,row['TYPE 1'])
            if not pd.isnull(row['TYPE 2']):
                effect += effectiveness(type2,row['TYPE 2'])
        if effect in result:
            result[effect] += 1
        else:
            result[effect] = 0
    return result


Venusaur = effects('Grass','Poison')
Blastoise = effects('Water','')
Charizard = effects('Fire','Flying')

Conclusion¶

There are several factors necessary to really become the very best like no one ever was in Pokémon. Choosing your starter is a bit more than just picking the one with the best stats. You must know how its typing matches up against the Pokémon in the rest of the game and understand your own battle style. As the Pokémon universe infinitely expands, it is important to keep track of how each generation is different and how to leverage new and old Pokemon to be a competitive Pokémon trainer. If you would like to dive deeper into advanced competitive Pokémon, I suggest researching battle strategies. These vary trememdously by player, so it is not something that can easily be researched with statistical data. Good luck catching them all!
